备注

Component 由引用封送是公共语言运行时中的所有组件的基类。 Component 可以进行远程处理，并且派生 MarshalByRefObject 类。 Component 提供的实现 IComponent 接口。 MarshalByValueComponent 提供的实现 IComponent ，按值封送。

您可以托管 Component 在实现任何对象 IContainer 接口，并且可以查询和获取从其容器的服务。 容器将创建 ISite 为每个 Component 它包含。 容器使用该站点来管理 Component ，并由 Component 与其容器进行通信。

一个 Component 应通过调用显式释放资源其 Dispose 方法，而无需等待自动内存管理通过隐式调用 Finalize 方法。 当 Container 将被释放中的所有组件 Container 同时会释放。
